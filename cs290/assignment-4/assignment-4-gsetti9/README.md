# CS290 Assignment 4 - Benny's List App

**Description:**  
This project implements a fully templatized version of the Benny's List app using Express.js and EJS. It features dynamic rendering of posts from JSON data, server-side routing, and client-side post generation.

**Skills Demonstrated:**  
- Express.js routing  
- EJS templating and includes  
- Client-side JS for dynamic post creation  
- JSON data handling
