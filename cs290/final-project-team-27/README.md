## Team 27 Benny's Quizmaster

Welcome one and all to Benny's Quizmaster!

This web application allows the user to create quizzes, for any and all subjects,
and allows for the creation of multiple choice and text answer questions.

Users can also search for and take other user's quizzes. After a quiz is taken,
a percentage of correct answers, as well as details on each question can be viewed.
